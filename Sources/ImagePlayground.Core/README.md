# ImagePlayground.Core

ImagePlayground.Core contains the core ImagePlayground image-processing APIs: image load/save, conversion, resize, crop, rotate, visual filters, text, watermarking, EXIF metadata, image comparison, thumbnails, icons, grids, mosaics, and GIF helpers.

This package remains available for compatibility with projects that intentionally reference the split assembly. New projects should usually reference the main `ImagePlayground` package instead.

## Supported Areas

- Image manipulation and transformation.
- Text and watermark overlays.
- Metadata and EXIF helpers.
- Image comparison.
- Base64 conversion.
- Avatar, icon, thumbnail, grid, mosaic, and GIF helpers.
