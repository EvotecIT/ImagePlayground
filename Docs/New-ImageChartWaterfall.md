# New-ImageChartWaterfall
## SYNOPSIS
Creates waterfall chart data item.

## SYNTAX
### __AllParameterSets
```powershell
New-ImageChartWaterfall [-Name] <string> [-Value] <double[]> [-Labels <string[]>] [-Color <ChartColor>] [<CommonParameters>]
```

## DESCRIPTION
Creates waterfall chart data item.

## EXAMPLES

### EXAMPLE 1
```powershell
New-ImageChartWaterfall -Name 'Change' -Value 120,-24,42 -Labels Start,Churn,Expansion
```
